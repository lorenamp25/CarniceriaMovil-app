


# 🛒 Carnicería Móvil App
<p align="center">
  <img src="./preview.png" width="300"/>
</p>
Aplicación web tipo e-commerce inspirada en plataformas como Uber Eats, desarrollada con React y optimizada para dispositivos móviles.

🔗 **Demo online:**  
https://carniceria-movil-ohllx9kqx-lorenas-projects-17e1e0a4.vercel.app/

---

## 🚀 Funcionalidades

- 📦 Visualización de productos
- 🔍 Buscador en tiempo real
- 🧭 Filtro por categorías
- 🛒 Carrito dinámico
- ❌ Eliminación de productos del carrito
- 💰 Cálculo automático del total
- 📱 Diseño responsive tipo app móvil
- ✅ Simulación de pedido con animación

---

## 🧠 Tecnologías utilizadas

- React (useState)
- JavaScript (ES6)
- Tailwind CSS
- HTML5 / CSS3

---

## 🎯 Objetivo del proyecto

El objetivo de este proyecto es simular una experiencia real de compra en una tienda online, enfocándose en:

- Diseño tipo aplicación móvil
- Experiencia de usuario (UX)
- Interacciones simples y rápidas
- Flujo de compra intuitivo

---

## 📱 Diseño

La interfaz está inspirada en aplicaciones como:

- Uber Eats
- Glovo

Priorizando:
- Navegación con una mano
- Elementos accesibles
- Simplicidad visual

---

## ⚠️ Nota

Este proyecto es una simulación frontend.

No incluye backend en producción, por lo que:
- Los pedidos no se almacenan
- Algunas funciones son simuladas
